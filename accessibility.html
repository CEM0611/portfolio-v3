HTML and CSS

This website was built from basic HTML tags and elements with Basic CSS involved as well. Colors were chosen with Google's built-in hex color picker for its search engine and HTML/CSS knowledge was obtained from w3schools resources and my COP 1500 course

Images and Vectors

The picture on the left on the main page is my portrait. The gradients and vectors were made using imported code from a vector I've created myself with InkScape and the gradient was created with JavaScript.

Navigation and Links
